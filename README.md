# AppLogs-Azure-Storage-Function
This repository has Site24x7 Azure functions for collecting logs from Microsoft Azure Blob Storage along with Azure ARM templates for automated deployment.

| Function | Description | Deployment |
|---|---|---|
| [Site24x7 Blob Logs Function](Blob_Logs) | Collects and forwards logs from Azure Blob Storage to Site24x7 AppLogs. | [![Deploy to Azure](https://aka.ms/deploytoazurebutton)](https://portal.azure.com/#create/Microsoft.Template/uri/https%3A%2F%2Fraw.githubusercontent.com%2Fsite24x7%2Fapplogs-azure-storage-function%2Fmaster%2Fdeployment%2Fsite24x7-azure-blob-logs-deployment.json) |
